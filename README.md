
# Generate Leads Backend

## Tech Stack

- **Framework**: [NestJS](https://nestjs.com/)
- **ORM**: [Prisma](https://www.prisma.io/)
- **DB**: PostgreSQL
- **Queue (planned)**: Redis + BullMQ
- **AI**: OpenAI SDK (`gpt-4o-mini`)
- **Auth**: Passport (Facebook strategy + JWT)
